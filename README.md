**Weather-App**

**ABOUT THE PROJECT:**

This projects can show you either weather of your live location or any custom city. 

**TECHNOLOGIES USED:**
*  This is a project made using html, scss, and vanilla javascript. 
* I have used weather and Html geolocation apis as well.
